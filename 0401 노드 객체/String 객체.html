String 객체

String 객체는 문자열을 손쉽게 만들고 다룰 수 있도록 합니다.
const str = 'javascript';
const str = new String('javascript');
console.log('문자길이:' + str.length) length : 문자열의 길이를 저장 ( str.length ) indexOf() : 매개변수로 전달된 문자열이 처음 등장하는 위치를 리턴 charAt() : 매개변수로 전달된 인덱스에 위치한 문자를 리턴 includes() : 매개변수로 전달된 문자열이 포함되어 있는지 여부를 리턴 substring() : 매개변수로 전달된 시작 인덱스부터 종료 인덱스 바로 앞까지 문자열을 추출하여 리턴 substr() : 매개변수로 전달된 시작 인덱스부터 매개변수로 전달된 개수만큼 문자열을 추출하여 리턴 split() : 매개변수로 전달된 구분자를 기준으로 문자열을 나눈 후 하나의 배열에 저장 replace() : 원본 문자열을 매개변수로 전달된 문자열로 치환 trim() : 문자열의 앞뒤 공백을 제거 toUpperCase() : 문자열의 영문자 중 소문자를 모두 대문자로 변환 toLowerCase() : 문자열의 영문자 중 대문자를 모두 소문자로 변환
